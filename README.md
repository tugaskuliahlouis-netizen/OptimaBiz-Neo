# OptimaBiz-Neo
OptimaBiz Neo adalah aplikasi web berbasis PWA (Progressive Web App) yang dirancang khusus untuk pemilik UMKM Indonesia. Tagline-nya: "Aksi Nyata, Bukan Teori" — karena fokusnya bukan sekadar menampilkan data, tapi mendorong pemilik bisnis untuk langsung mengambil tindakan nyata berdasarkan data tersebut.
